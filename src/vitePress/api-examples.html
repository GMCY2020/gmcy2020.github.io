运行时API示例 ​

这个页面演示了一些由VitePress提供的运行时api的用法.

主要的 useData() API 可以用来访问 站点(site), 主题(theme), 和 页面数据 对于当前页面. 这适用于 .md 和 .vue 文件:

md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## 结果

### 主题数据
<pre>{{ theme }}</pre>

### 页面数据
<pre>{{ page }}</pre>

### 主题数据页首页
<pre>{{ frontmatter }}</pre>
