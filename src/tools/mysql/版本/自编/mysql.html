模板

sql
create table tb_user (
     id int primary key auto_increment comment 'ID 唯一标识',
     username varchar(20) not null unique comment '用户名',
     name varchar(10) not null comment '姓名',
     age int comment '年龄',
     gender char(1) default '男' comment '性别'
) comment '用户表';

auto_increment 自动增长

约束

关键字约束描述
not null非空约束限制该字段值不能为null
unique唯一约束保证字段的所有数据都是唯一, 不重复的
primary key主键约束主键是一行数据的唯一标识, 要求非空且唯一
default默认约束保存数据时, 如果未指定该字段, 则采用默认值
foreign key外键约束让两张表的数据建立连接, 保证数据的一致性和完整性

数值类型描述字符串类型描述日期类型描述格式

mediumint大整数值tinyblob不超过255个字符的二进制数据year年份值YYYY

bigint极大整数值blob二进制形式的长文本数据timestamp混合日期和时间值，时间戳YYYY-MM-DD HH:MM:SS
float单精度浮点数值text长文本数据
double双精度浮点数值mediumblob二进制形式的中等长度文本数据
