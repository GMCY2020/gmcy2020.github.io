MySQL 数据库 之 SQL语句 ​

一. DDL ​

Data Definition Language

数据定义语言
用来定义数据库对象 (数据库, 表, 字段)

1. 数据库 ​

显示所有的数据库

sql
show databases;
1

创建数据库

sql
create database 数据库名;

create database if not exists 数据库名;
1
2
3

使用和切换数据库

sql
use 数据库名;
1

查看当前正在使用的数据库

sql
select database();
1

删除数据库

sql
drop database 数据库名;

drop database if exists 数据库名;
1
2
3

2.1 表 创建 ​

创建表

sql
create table 表名 (
    字段1 数据类型 [约束] [comment '字段1注释'],
    ...
    字段n 数据类型 [约束] [comment '字段n注释']
) [comment '表注释'];

2.2 表 查询 ​

显示所有表

sql
show tables;
1

查看指定的表结构

sql
desc tb_user;
1

查看数据库的建表语句

sql
show create table 表名;
1

2.3. 表 修改 ​

添加字段

sql
alter table 表名 add 字段名 类型(长度) [comment '注释'][约束];
1

修改字段类型

sql
alter table 表名 modify 字段名 新数据类型(长度);
1

修改字段名和字段类型

sql
alter table 表名 change 旧字段名 新字段名 类型(长度) [comment '注释'][约束];
1

删除字段

sql
alter table 表名 drop column 字段名;
1

修改表名

sql
rename table 表名 to 新表名;
1

删除表

sql
drop table if exists 表名;
1

二. DML ​

Data Manipulation Language

数据操作语言
用来对数据库表中的数据进行 增删改

1. insert ​

指定字段添加数据

sql
insert into 表名(字段1, 字段2) values (值1, 值2);
1

全部字段添加数据

sql
insert into 表名 values (值1, 值2, ...);
1

批量添加数据(指定字段)

sql
insert into 表名(字段1, 字段2) values (值1, 值2), (值1, 值2);
1

批量添加数据(全部字段)

sql
insert into 表名 values (值1, 值2, ...), (值1, 值2, ...);
1

2. update ​

修改数据

sql
update 表名 set 字段1 = 值1, 字段2 = 值2, ... [where 条件];
1
  • 没有where将会修改所以数据

3. delete ​

删除数据

sql
delete from 表名 [where 条件];
1

1. 基本查询 ​

查询多个字段

sql
select 字段1, 字段2, 字段3 from 表名;
1

查询所以字段(通配符)

sql
select * from 表名;
1

设置别名

sql
select 字段1 [as 别名1], 字段2 [as 别名2] from 表名;
1

去除重复记录

sql
select distinct 字段列表 from 表名;
1

2. 条件查询 ​

sql
select 字段列表 from 表名列表 where 条件列表;
1

3. 分组查询 ​

sql

4. 排序查询 ​

sql

四. DCL ​

Data Control Language

数据控制语言
用来创建数据库用户, 控制数据库的访问权限

五. 注 ​

1. 约束 ​

关键字约束描述
not null非空约束限制该字段值不能为null
unique唯一约束保证字段的所有数据都是唯一, 不重复的
primary key主键约束主键是一行数据的唯一标识, 要求非空且唯一
default默认约束保存数据时, 如果未指定该字段, 则采用默认值
foreign key外键约束让两张表的数据建立连接, 保证数据的一致性和完整性

2. 数据类型 ​

数值类型描述字符串类型描述日期类型描述格式
tinyint小整数值char定长字符串date日期值YYYY-MM-DD
smallint大整数值varchar变长字符串time时间值或持续时间HH:MM:SS
mediumint大整数值tinyblob不超过255个字符的二进制数据year年份值YYYY
int大整数值tinytext短文本字符串datetime混合日期和时间值YYYY-MM-DD HH:MM:SS
bigint极大整数值blob二进制形式的长文本数据timestamp混合日期和时间值，时间戳YYYY-MM-DD HH:MM:SS
float单精度浮点数值text长文本数据
double双精度浮点数值mediumblob二进制形式的中等长度文本数据
decimal小数值(精确值高)mediumtext中等长度文本数据
longblob二进制形式的极大文本数据
longtext极大文本数据

3. 运算符 ​

比较运算符功能逻辑运算符功能
>大于and 或 &&并且(多个条件同时成立)

<小于not 或 !非, 不是

<> 或 !=不等于

in(...)在in之后的列表之中的值, 多选一

is null是null
