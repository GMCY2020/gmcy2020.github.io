1. Flowchart ​


介绍 ​


流程图
Mermaid 的基础语法
重点有 节点形状 和 箭头类型

1.1 一个节点(默认) ​

代码

```mermaid
flowchart LR
    id
```

预览

mermaid
flowchart LR
    id
1
2

1.2 一个带有文本的节点 ​

代码

```mermaid
flowchart LR
    id1[文本在盒里面]
```

预览

mermaid
flowchart LR
    id1[文本在盒里面]
1
2

1.3 图 ​

代码

```mermaid
flowchart TD
    开始 --> 结束
```

```mermaid
flowchart LR
    开始 --> 结束
```

参数

flowchart                           流程图

TB(Top Bottom) 或者 TD(Top Down)    箭头方向从上到下
BT                                  箭头方向从下到上
LR(Left Right)                      箭头方向从左到右
RL                                  箭头方向从右到左

预览

mermaid
flowchart TD
    开始 --> 结束
1
2
mermaid
flowchart LR
    开始 --> 结束
