1. 配置 Maven ​

1.1 下载安装 ​

  • 官网下载 3.6.3 版本 zip

  • 解压到相关路径

    • 这里解压到 D:\Depends\Apache\apache-maven-3.6.3

    • 路径尽量使用英文

  • 并且添加文件夹 mvn_repo 作为 仓库

1.2 修改配置文件 ​

  • 修改 D:\Depends\Apache\apache-maven-3.6.3\conf\setting.xml

  • 在 <setting> ... </setting> 添加本地仓库地址 第55行左右

    xml
    <!-- 本地仓库地址 -->
<localRepository>D:\Depends\Apache\apache-maven-3.6.3\mvn_repo</localRepository>
    1
    2

  • 在 <mirrors> ... </mirrors> 中 添加阿里云镜像

    xml
    <!-- 阿里云仓库 -->
<mirror>
  <id>almaven</id>
  <name>aliyun maven</name>
  <url>https://maven.aliyun.com/nexus/content/groups/public</url>
  <mirrorOf>central</mirrorOf>    
</mirror>

1.3 添加环境变量 ​

右键 此电脑 -> 单击 属性 -> 单击 高级系统设置 -> 单击 环境变量 -> 选择 Path -> 单击 编辑 -> 单击 新建 -> 添加 maven的bin路径 -> 一直确认

1.4 检验 ​

  • win+r 调出 cmd

  • cmd 输入 mvn help:system 初始化

  • cmd 输入 mvn -version

1.5 idea 配置 maven ​

  • 打开 idea 首页 -> 自定义 (Customize) -> 所有设置 (All settings...)

  • 构建、执行、部署 (build, Execution, Deployement) -> 构建工具 (Build Tools) -> Maven

  • 配置3个路径
